# BaileyDB

BaileyDB is a React JS website with a Node JS back end made by Sam Bailey to allow users to browse and create a watch list whilst using the TMDB API.

## Built with:

- React JS
- React Bootstrap
- Node JS
- Postgres Database
- Auth0
- TMDB API
- Microsoft Azure Hosting Server
